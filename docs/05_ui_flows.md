# UI Flows

## Admin /admin
- Dashboard for seeing the lesson plans at a glance.
- Options to update lessons
- Navigation to add/update venues, event-types, plan lessons.

## Admin: plan lesson /admin/plan-lesson?...
- Opened from an unplanned virtual occurrence
- Fields are editable:
  - date
  - duration
- Save:
  - Creates the Event document
  - Redirects to Event Planner for lesson entry
